 # 📊 Student Data Analysis (Pandas Project)

## 📌 Overview

This project performs analysis on student data using Python and Pandas.

## ⚙️ Features

* ✔ Add Result column (Pass/Fail)
* ✔ Grade classification (A/B/C)
* ✔ Average marks per branch
* ✔ Best performing branch
* ✔ City-wise student count
* ✔ Pivot table analysis
* ✔ Top performers filtering
* ✔ Multi-condition filtering

## 🛠 Tools Used

* Python
* Pandas
* Jupyter Notebook

## 📂 Files

* `student_analysis.ipynb` → Main notebook
* `student_analysis.csv` → Output dataset

## 🚀 How to Run

1. Open the notebook in Jupyter Notebook / JupyterLab
2. Run all cells

## 📈 Sample Insights

* Identified top-performing students
* Compared branch performance
* Analyzed city-wise distribution
